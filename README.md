# Andrew Anil George - Portfolio

Personal portfolio website built with modern frontend technologies.

## Tech Stack
- Next.js
- TypeScript
- Tailwind CSS
- Framer Motion

## Features
- Responsive layout (desktop + mobile)
- Scroll-based section navigation
- Light and dark mode
- Resume download
- Clean, minimal design focused on readability

## Live Site
Deployed on Vercel.

## Running Locally
```bash
npm install
npm run dev
